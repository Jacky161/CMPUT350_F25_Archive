# Lab 1 prep

See full instructions [here](https://cmput350.danielzhang.net/labs/lab1/prep/lab1_prep.html).

Implement the procedures operating on `Matrix`
in `prep.c`.
Then, write code in `main` to test your 
implementations.

Compile with 
```bash
gcc -g -Wall -Wextra -Wconversion -Wsign-conversion -O -std=c99 prep.c
```
Make sure that your code compiles without errors, nor warnings.
Use `valgrind` to ensure your implementation doesn't leak memory.
